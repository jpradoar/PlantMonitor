<div align="center"><h3> El material utilizado </h3> </div>

Realmente no se cual es el tipo de material, aunque es un plástico lo bastante resistente para aguantar cierta presión y torsión, impermeable y bastante liviano (lo saque de un cartel que había la basura). 

Para amurar, simplemente hice un agujero en la pared y atornille directamente el plástico a la misma, le puse una arandelita para no marcar ni romper el plástico. Luego lo dejé algunos días con algo de peso dentro para ver si se movía con el viento, por suerte quedo perfecto (incluso hubo dos días de lluvia y cero problemas).

<br> 

<div align="center"><h3> Caja protectora </h3> </div>

Para la caja utilice el mismo material, cortando la estructura y pegando con semento de contacto. El tamaño es un poco grande para todo el proyecto, pero trate de que entre el protoboard grande + la batería + alguna que otra cosa que necesite dejar dentro. 

Le arme una tapa superior para que sea mas fácil poner y sacar las cosas.  Las bisagras no están atornilladas por que el tornillo partía el plástico y seguramente se podía llegar a falsear, así que decidí pegarlas con el mismo pegamento.  Al igual que toda la estructura. 
Finalmente le coloque un sellador de silicona para evitar filtraciones. 

Lo deje secar un día completo al sol y para hacer una prueba le puse agua adentro. Paso la prueba sin problemas. :)

<div align="center"><h3> Veleta</h3> </div>

Para hacer la veleta me base en el diseño de la cola de un avión. (me aburre el típico gallito).
Lo corte con el cúter y con una lija (de 200) le comí los bordes superiores y las puntas frontales para que tenga una mejor aerodinámica y un acabado mas lindo y prolijo, la parte de abajo la deje cuadrada y sin bordes redondeados.
